# 使用GAN生成新的游戏角色

## 摘要
Generative Adversarial Networks（简称GAN），中文名叫生成对抗网络。我们将使用它，来生成新的游戏角色。

## 依赖 （pip install）

```
cv2
tensorflow( >=1.0)
scipy
numpy
```
## 使用方法

```
cd yysGAN
python yysGAN.py
```
